# Contributing to WinFast Pro X

Thank you for your interest in contributing!

## How to contribute

### Bug reports
Open an issue with:
- Windows version and build number
- Steps to reproduce
- Expected vs actual behavior
- Log file if available (`%APPDATA%\WinFastProX\logs\`)

### Feature requests
Open an issue labeled `enhancement` with:
- Clear description of the proposed feature
- Use case / who benefits from it
- Any reference to Windows internals documentation if applicable

### Tweak research
If you have documented evidence of an optimization tweak that produces measurable results:
- Include benchmark methodology
- Include rollback procedure
- Reference the Windows component affected (registry path, service, driver)
